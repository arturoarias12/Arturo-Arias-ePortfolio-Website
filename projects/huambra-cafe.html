Context

Huambra Café is a coffee shop startup based in Pucallpa, Peru, a city in the heart of the Amazon jungle. As part of a broader modernization effort, the business wanted structured visibility across all key operational dimensions: revenue, expenses, customer traffic, and inventory and supply stock. All records were maintained in a native local database, with no analytics layer and no systematic way to monitor performance over time.

The goal was to migrate that data to the cloud, build a reliable automated update pipeline, and deliver a self-service dashboard the owner could use independently for daily monitoring and strategic decisions.

What was built

The project covered the full data stack, from cloud migration to a live, auto-updating BI product in production:

  • Database migration: the native operational database was migrated to Google Cloud, with BigQuery established as the central analytics data warehouse
  • Automated data pipeline: a Python job deployed on Cloud Run handles incremental data updates, managing the complete processing and transformation pipeline to keep the warehouse current
  • Power BI dashboard: built entirely from scratch, including the star schema data model, all DAX measures, and the report views, covering revenue, expenses, customer traffic, and product and supply inventory
  • Production deployment: the finished dashboard was published to Microsoft Fabric with auto-refresh configured, handed off to the owner ready for day-to-day strategic decision making
